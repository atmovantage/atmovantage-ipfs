What Can You Do With An IPFS Website?

Austin Shackelford

Well the first thing you can do is host a blog like this. Building the blog using an offline CMS tool like Publli and then publishing via a GitHub repository is very easy to setup. This particular blog is written using a block-style editor, very similar to the experience users are familiar with from WordPress' Gutenberg.

You can embed HTML and thus insert YouTube videos and other embeddable content, such as the video above.

You can insert a photo gallery like the one I made above from my exploits in a game called No Man's Sky.

You can even embed javascript code, such as the contact form above!

Although static sites might seem limiting compared to the dynamic websites we've become accustomed to, static sites hosted on a decentralized, peer-to-peer network like IPFS have many advantages that can make it worthwhile. I'll cover some of those advantages in more detail in another post.
